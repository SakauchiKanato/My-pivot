  • タイムラインデータおよび作成・更新処理をログインユーザーに紐付ける形に更新
  • ログイン・新規登録の切り替えができるモダンな認証画面の構築

2. ⏳ 五里霧中モード & タイムラインの安定化

以前に発生していた画面のクラッシュやパスズレを解消しました。

  • ピボット取得時にリレーション先のタグ配列がシリアライズされずクラッシュしていた問題を修正

新規機能

3. 📚 地球の書庫 v3 の完全移植

提示された my_pivot_v3.html の全ロジックとデザインをReactコンポーネントとして完全に移植しました。

  • 本棚の立体的な配置や、本を開くペラペラとした3Dめくりアニメーションの再現
  • 入力に応じて過去の悩みを引っ張ってくる「召喚 (Recall)」機能のReact同期

  • 結果をたずねるベルマーク通知による、タイムライン上のスクロールジャンプ機能
  • デモ用として localStorage を使って安全かつ独立して動くように設計

💻 ローカル環境での起動方法

リポジトリを最新に更新し、以下の2ステップで起動します。

Step 1: バックエンドの起動 (Python / FastAPI)

  1. ターミナルで backend フォルダに移動し、必要なパッケージをインストールします： 
    cd backend
pip install -r requirements.txt
  2. DBにデモ用のユーザーとタイムラインデータを登録するスクリプトを動かします（DB初期化）： 
    PYTHONPATH=. python seed.py
  3. FastAPIサーバーを起動します： 
    uvicorn app.main:app --reload
⚠️ Port 8000 がすでに使用されているエラーが出た場合
バックエンドの古いプロセスが終了せず残っています。以下のコマンドでプロセスを終了して再実行してください：
  • Mac: kill -9 $(lsof -t -i:8000)
  • Windows: taskkill /f /im python.exe

Step 2: フロントエンドの起動 (TypeScript / React / Vite)

  1. 別の新しいターミナルを開き、frontend フォルダに移動してパッケージをインストールします： 
    cd frontend
npm install
  2. 開発用サーバーを立ち上げます： 
    npm run dev
    これでブラウザが自動的に開き、http://localhost:5173 等でアプリを確認できます。
