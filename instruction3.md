# 【AI実装指示用プロンプト】呼吸する紙とインクの記憶（Human-Touch Book UI）

## 📋 役割と目的

あなたは世界的なデザイン賞（Apple Design Award等）を総なめにしてきた、伝説的なUI/UXデザイナー兼シニアエンジニアです。
ユーザーから提供された見開き本スタイルのUI画面（スクリーンショット 2026-07-09 17.06.49.jpg）をベースに、**デジタルの冷徹さを排除し、人間の温かみ、迷い、時間の経過を感じさせる「人間らしい（Human-Touch）」動的アニメーションUI**を実装するための設計とソースコード（React, Tailwind CSS, Framer Motion）を出力してください。

単に整然と並んだデータを表示するのではなく、ユーザーの「人生の軌跡」が紙に染み込んでいくような、エモーショナルでクラフトマンシップ溢れる体験を構築します。

---

## 📜 コンセプト：「インクの鼓動と、呼吸する紙（Ink, Breath, & Craftsmanship）」

- **背景世界観:** この画面は、システムが生成したログではない。人間が悩み、決断した「魂のノート」である。そのため、UI全体が「まるで呼吸しているかのような有機的な揺らぎ」を持つ。
- **紙とインクの質感:** 画面上のテキストやバッジ（「後悔が残る」「判断は妥当だった」等）は、ただレンダリングされるのではなく、万年筆のインクが紙にじわっと吸い込まれ、馴染んでいくような人間らしいエフェクトを伴う。
- **時間の経過（経年変化）:** 過去の古い決断（過去の年表）ほど、紙の端がかすかにセピア調に変化していたり、何度も見返した記録は「手垢が馴染む」ようにフォントやシャドウが柔らかくなる。

---

## 🎨 ハッカソンで勝つための「人間らしい」動的演出（AIへの要求事項）

LLMは以下の4つのインタラクション・演出を完全に満たすUIコンポーネントを実装してください。

### 1. タブ切り替え時の「リアルなページめくり」 (Physical Page Flip)

- **仕様:** 左上の「目次」「書く」「探す」「年表」のタブを切り替える際、画面がパッと切り替わるのではなく、**本物の紙のページをめくるように、滑らかな3Dの歪み（CSS 3D Transforms）と、めくれた紙の裏に落ちる繊細なシャドウ**を表現する。
- **演出:** 右側の「この本の年表」ページも、左の操作に連動してかすかに風で揺れるような、微細な「紙のしなり（Web Audioや物理演算風のSpring）」を表現。

### 2. インクの染み込みと手書きマーカー (Ink Bleeding & Hand-drawn Highlights)

- **仕様:** 画面内の重要ステータス（「見込み★3」「後悔が残る」など）がロードされた瞬間、またはホバーした瞬間、**蛍光ペンでシュッと手書きで線を引いたようなマーカーアニメーション（SVGのstroke-dashoffset操作）**が走る。
- **演出:** テキストが出現する際、一瞬だけフォントの輪郭が「インクが滲むように」微かにぼやけ（CSS blur）、100msかけてカチッと定着するような、インクの生々しさを表現。

### 3. 感情に応じた「呼吸するカード」 (The Breathing Component)

- **仕様:** 『スクリーンショット 2026-07-09 17.06.49.jpg』にあるような「大学の勉強をすべきか否か」というユーザーの迷いに対して、カード自体が生き物のように優しく、不規則に「呼吸（ゆっくりとしたScaleやOpacityの超低頻度パルス：4000ms〜6000ms周期）」する。
- **演出:** 「後悔が残る」といったネガティブ・葛藤系のカードは、心拍数が少し乱れるような微細な震え（微弱な揺らぎ）を持ち、「判断は妥当だった」という解決済みのカードは、深く落ち着いた静かな呼吸をする。

### 4. 人間の手による「書き込み」UX (Human Hand Jotting)

- **仕様:** 「この本に書く」や「追記する」ボタンを押した際、入力フォームが無機質に出現するのではなく、**ノートの余白に鉛筆でサラサラと枠線が描かれていくような線画アニメーション**と共に展開する。

## 🛠 技術スタック（指定）

- **Frontend:** React, TypeScript
- **Styling:** Tailwind CSS (背景は `スクリーンショット 2026-07-09 17.06.49.jpg` にある、目に優しい最高品質の高級紙（インボイス／ブックアイボリー）のカラー `#FDFBF7` 前後をベースとし、テクスチャ感を持たせる)
- **Animation:** Framer Motion (3Dフリップ、SVGの手書きライン描画、カードの呼吸モーションを制御)

---

## 🎯 出力フォーマット

1. **Human-Touch Interaction Design:** 人間の温かみ（アナログ感）をデジタルで最速（60fps）かつ軽量に表現するためのCSS/SVGハックの解説。
2. **Tailwind Custom Theme:** 紙の質感、インクの黒（完全な黒ではなく、少し青みがかった、またはセピアがかった墨色）、エモーショナルなカラーパレットの定義。
3. **The Masterpiece Code:**
   - `BookLayoutContainer`（見開きのページめくり・しなりを実現するコンポーネント）
   - `InkBleedingBadge`（手書きマーカーとインクの染み込みを再現したステータスタグ）
   - `BreathingLogCard`（感情の揺らぎ・呼吸をアニメーション化するカード）
4. **Hackathon Judging Hack:** 審査員がプレゼン（デモ）の画面を見た瞬間に、「このプロダクトには愛と魂がある」と感じて最高得点を投じたくなるような、デモ映えする操作手順とイージングの秒数設定のアドバイス。
