# 【AI実装指示用プロンプト】知の樹木と開花する書庫（Botanical Archive & Hero Transition）

## 📋 役割と目的

あなたは世界最高峰のUI/UXインタラクションデザイナーであり、シリコンバレーのトップテック企業で活躍するシニアフルスタックエンジニア（Tech Lead）です。
ユーザーから提供されたタイムラインUIをベースに、**「書庫（本棚）」という静的な空間に「植物の成長（オーガニック・ボタニカル）」という生命力を融合させた、極上の動的インタラクションUI**を実装するための設計とソースコード（React, Tailwind CSS, Framer Motionを想定）を出力してください。

単なるデータリストをレンダリングするのではなく、ユーザーが「触るだけで五感が満たされる」ような、エモーショナルで一目惚れする体験（Experience）を構築することが目的です。

---

## 🌿 コンセプト：「記憶の樹脈と、開花する本（The Botanical Archive）」

- **背景・世界観:** ユーザーの過去の決断や記録が蓄積される空間。これは無機質なログではなく、ユーザー自身が育てる「知の樹木」である。
- **タイムラインの構造:** 中央の軸は直線ではなく、微かにうねりを持つ「蔓（つる）や木の幹」。そこから伸びるノードは「蕾（つぼみ）」であり、記録カードは生命力を持った「葉」として表現される。
- **本へのアクセス:** 葉（記録カード）をタップすると、それが一枚の葉から一冊の「重厚な本」へとシームレスに変容（モーフ）し、読書空間（詳細画面）へとユーザーを誘う。

---

## 🎨 動的アニメーション・詳細仕様（AIへの要求事項）

LLMは以下の4つのインタラクション・シーケンスを完全に満たすUIコンポーネントおよびアニメーションロジックを設計・実装してください。

### 1. ページロードとスクロール（The Sprouting & Growth）

- **中央の軸（幹・蔓）:** ページスクロールの速度や深度に連動し、上から下へ滑らかに「伸びていく（SVGパスの描画アニメーションなど）」。
- **ノード（蕾）:** スクロールによって視野に入った瞬間、幹からプクッと蕾が膨らむようにスケールアップ（Spring物理モーション）して出現。
- **カード（葉）:** 蕾が弾けるように、カードが微かに回転と独自のイージングを伴ってふわりと展開（RotateZ: 5deg -> 0deg, Fade-in）。

### 2. マウスホバー（Breeze of Wisdom）

- **カード（葉）のフローティング:** カードにカーソルを合わせると、まるで見えない心地よい風が吹いたかのように、カード全体がゆっくりとフワッと上下に揺れる（Y-axis: -4px, Rotate: 0.5degの無限ループ）。
- **養分の流動（パルス効果）:** ホバー時、中央の幹からノードを経由して、対象のカードへと微弱な「光の波（グラデーションのパルス）」がスッと流れ込み、カードの輪郭が微かに発光する。
- **「この本を開く」リンク:** テキストが有機的な緑、またはプライマリーカラーで発光し、矢印アイコンが前へ前へと連続的に押し出されるようなインダクティブなモーション。

### 3. 【最重要】タップ時の画面遷移（Blooming into the Book）

- **共有要素遷移（Shared Element Transition）の実行:**
  - ユーザーがカード（葉）をタップした瞬間、他のすべてのカードと背景のタイムライン軸は、ピントがボケて（Blur）奥へと静かに沈み込む（Scale: 0.95, Opacity: 0.4）。
  - タップされたカードだけが、画面の制約を無視してレイアウトを維持したまま、**画面中央へとダイナミックに浮き上がりながら拡大**。
  - 拡大の最中、カードの質感がフラットな「葉」から、立体的な影を持つ「本の表紙」へとシームレスに変容（Morphing）。
  - 画面中央に到達した瞬間、その**本の表紙がフワリとめくれ上がり、中の白いページ（紙のテクスチャ）が画面全体を美しく覆いつくして**、詳細画面（本の本文）へ滑らかにコンポーネントが切り替わる。
  - _※Reactの `Framer Motion` における `layoutId`（FLIPアニメーション）を用いた、ページを跨ぐレイアウト共有技術を必ず使用すること。_

### 4. 状態表現のオーガニック化（例：NGバッジなど）

- **棘（とげ）と枯れ葉:** 画面内の「NG」などのネガティブなステータスバッジは、単なる赤い丸ではなく、バラの「棘」や「紅葉・枯れ葉」を連想させる有機的なデザイン・色調にする。
- **アラートモーション:** ホバー時には、植物が身を守るような「カサッ」という乾いた質感を表現する微細な左右の震え（X-axis high-frequency jiggle）と、濃い真紅のシャドウが背後に落ちる演出を施す。

## 🛠 技術スタック・実装アプローチ（指定）

- **Frontend:** React (Next.js または Vite), TypeScript
- **Styling:** Tailwind CSS (ダークモードを基調とした、深みのあるマホガニー・アンバー・ディープボタニカルグリーン・Glassmorphismの融合)
- **Animation:** Framer Motion (必須。`layoutId` を使用したShared Element Transition、および `AnimatePresence` によるページめくり演出)
- **Icons / SVG:** 幹や蔓の表現には動的なSVG path操作（`pathLength` 等）を使用
