# My Pivot

> 迷った数だけ、私は進んだ。五里霧中のロードマップを、確かな軌跡に変える。

過去の選択・迷いを記録し、次の意思決定のときに「過去の自分」を参考人として参照できるアプリ。

---

## これは何のプロジェクト？

**ターゲット**：重要な選択を繰り返す学生・院生
**課題**：「なぜその選択をしたか」の感情・文脈が記憶から消え、次の選択でゼロからやり直してしまう
**価値**：迷いをそのまま記録 → 自動でタグ整理 → タイムラインで振り返り、の1ループを提供する

---

## 技術構成

Python を主軸に、フロントだけ React を使う構成。

| 領域 | 技術 | 言語 |
|------|------|------|
| フロントエンド | Vite + React | TypeScript |
| バックエンド | FastAPI | **Python** |
| タグ整理ロジック | ルールベース（将来LLM追加可） | **Python** |
| DB | SQLite + SQLModel | **Python** |

> なぜこの構成かは `構成の選び方.md` を参照。

---

## セットアップ手順

### 1. バックエンド（Python）

```bash
cd backend

# 必要なライブラリをインストール
pip install fastapi "uvicorn[standard]" sqlmodel

# デモ用データを投入（初回のみ）
python seed.py

# サーバー起動
uvicorn app.main:app --reload
```

起動後、http://localhost:8000/docs でAPIドキュメントを確認できる。

### 2. フロントエンド（React）

別のターミナルで：

```bash
cd frontend

# 必要なパッケージをインストール
npm install

# 開発サーバー起動
npm run dev
```

起動後、http://localhost:5173 でアプリが開く。

> ⚠️ バックエンドとフロントエンドの**両方**を起動しておく必要がある。
