UI Update Summary
タイムライン機能追加と「Human-Touch」UIへの刷新

チームメンバー各位

今回のアップデートでは、ユーザーの記録と決断をよりエモーショナルに、そして直感的に体験できるように、UI/UXを大幅に刷新しました。単なるデータの羅列ではなく、**「魂のノート」「記憶の樹脈」**というコンセプトに基づき、有機的で温かみのあるデザインを取り入れています。

1. 新機能: The Botanical Archive（タイムライン機能）

過去の決断と結果を振り返る「タイムライン機能」を実装しました。無機質な直線ではなく、「知の樹木」が育っていく過程を表現しています。

🌱 デザインコンセプト

  • 成長する幹: スクロールに合わせて伸びていく有機的なSVGの幹。
  • 蕾（つぼみ）と葉: 記録のノードを「蕾」、詳細カードを「葉」に見立て、画面に入る瞬間に弾けるように出現。
  • 書庫バッジの可視化: 「私の書庫🏛️」「共同の書架🌿」「先達の書架📜」が一目でわかるグラスモーフィズムデザインのバッジを追加。
  • 可視化するもの: 「タイトル」「日付」「書庫の種類」＋「簡単な結果ステータス（うまくいった/後悔が残るなど）」

✨ インタラクション

  • 微風のホバーエフェクト: カードにカーソルを合わせると、見えない風が吹いたようにフワッと浮遊し、微小な角度で揺れます。
  • パルス発光: ホバー時、カードの縁がゆっくりと脈打つように光り、ユーザーの視線を誘導します。
  • 【重要】本へのシームレスなジャンプ: タイムライン上のカードをタップすると、その記録が属する「本」の詳細画面へ一瞬でワープ（遷移）する機能を実装しました。

2. 既存UIの改良: Human-Touch Book UI

「この本」を開いた時の体験を強化しました。デジタル感を排し、インクや紙、時間の経過を感じさせる「呼吸するUI」を取り入れています。

🖋️ インクの鼓動とマーカー

  • インクの染み込み: ステータスバッジ（「見込み★3」「うまくいった」等）が表示される際、パッと出るのではなく、インクが紙にじわっと滲んで定着するようなエフェクト（blur → sharp）を追加。
  • 手書きマーカー: バッジにホバーすると、蛍光ペンでサッと線を引いたような手書き風のアニメーションが走ります。

🫀 感情に応じた「呼吸するカード」

  • 状態表現のオーガニック化: ただの四角いカードではなく、カード自体が生きているように微細にアニメーションします。
  • 感情の揺らぎ: 「後悔が残る(NG)」カードは、枯れ葉が風で揺れるような不規則で細かい震え（微震え）を持たせました。
  • 穏やかな呼吸: 「うまくいった(OK)」カードは、5秒周期で深く落ち着いた静かな呼吸（影の伸縮）をします。

📖 ページと紙の質感強化

  • モードタブの立体感: 「目次」「書く」などのタブを切り替える際、ページをめくるようなわずかな弾み（Spring）と、アクティブ状態での沈み込み（シャドウの表現）を強化しました。
  • 紙の色の調整: 背景色を、目に優しい高級紙（インボイス/ブックアイボリー）を連想させるわずかに温かみのあるトーンへ微調整しました。

3. 検索・フィルタリングのUI/UX改善と高速化

ファーストビューの視認性を高め、大量の記録から目当ての決断を瞬時に見つけられるように、検索周りのUIと裏側の処理パフォーマンスを大きく改善しました。

🔍 検索UIのスマート化とアコーディオン表示

  • 詳細条件の折りたたみ: これまで横に広がり画面を圧迫していた詳細フィルター（タグ、結果、自信などの絞り込み）を、「▼ 詳細条件」の中に格納するアコーディオン形式に変更。ファーストビューで「書庫」が確実に見えるようになりました。
  • 縦積みレイアウトと余白の調整: 検索パネル全体のレイアウトを整理し、要素間の無駄な余白（gapやmargin）を詰めることで、よりコンパクトかつ直感的な操作感を実現しています。

⚡ 劇的なパフォーマンス改善（サクサク検索）

  • 検索処理のキャッシュ化: 検索ボックスに文字を1文字入力するたびに発生していた重い処理（全ての本のタグやテキストを結合する処理）を最適化。
  • 入力遅延の解消: 記録や本の数が増えても処理が重くならないよう、事前にテキスト情報をキャッシュ（一時保存）する仕組みを導入し、文字を打った瞬間にサクサク絞り込めるようになりました。

🌳 タイムライン文脈の可視化

  • 本のタイトル表示: タイムラインに表示される各記録（葉っぱ）に、それが「どの本の記録なのか（例：就活の記録）」を追加表示するようにしました。これにより、タイムライン上からでも記録の文脈が一目で理解できるようになっています。
